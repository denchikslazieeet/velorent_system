# Структура проекта

- `accounts/` — пользователи, роли;
- `catalog/` — велосипеды, тарифы, точки выдачи;
- `rentals/` — бронирования, аренды, платежи;
- `integrations/` — очередь событий синхронизации с 1С;
- `dashboard/` — веб-страницы личного кабинета и панели оператора;
- `api/` — REST API;
- `templates/` — HTML-интерфейсы;
- `static/` — стили;
- `docs/` — проектная документация.
