# Описание сущностей

## User
Пользователь системы с ролью:
- customer
- operator
- admin

## PickupLocation
Точка выдачи/приема:
- название;
- адрес;
- телефон;
- режим работы.

## Tariff
Экономические параметры аренды:
- ставка за час;
- ставка за сутки;
- залог;
- штраф за просрочку.

## Bike
Инвентарная единица велопарка:
- название;
- серийный номер;
- категория;
- тариф;
- текущая точка;
- статус.

## Booking
Заявка клиента на аренду:
- клиент;
- велосипед;
- период;
- стоимость;
- залог;
- статус.

## Rental
Фактическое исполнение брони:
- оператор выдачи;
- оператор приема;
- состояние при выдаче и возврате;
- штрафы;
- итоговая стоимость.

## Payment
Финансовая операция:
- тип платежа;
- метод оплаты;
- статус;
- сумма.

## SyncEvent
Событие интеграции:
- сущность;
- тип события;
- JSON-полезная нагрузка;
- статус обмена.
