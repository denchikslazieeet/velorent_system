# VeloRent Platform

Проектная и программная основа системы велопроката на базе **Django + Django REST Framework + PostgreSQL**  
Система спроектирована на основании курсовой работы **«Разработка комплекса программных приложений для велопроката»**, но с переосмысленной архитектурой, без использования старых макетов.

## Что входит в проект
- веб-приложение для клиентов;
- операторская панель для выдачи и возврата велосипедов;
- REST API для мобильного/внешнего клиента и интеграции с 1С;
- доменная модель аренды, бронирования, платежей и техсостояния;
- подготовленный слой интеграции с 1С;
- документация по архитектуре, бизнес-правилам и API.

## Основные роли
- **Клиент** — регистрация, просмотр каталога, создание брони, просмотр истории аренд.
- **Оператор** — подтверждение брони, выдача велосипеда, прием возврата, фиксация состояния, прием оплаты.
- **Администратор** — управление справочниками, тарифами, пользователями и контроль интеграций.

## Быстрый старт

### 1. Установка зависимостей
```bash
python -m venv .venv
source .venv/bin/activate  # Linux/macOS
# .venv\Scripts\activate   # Windows

pip install -r requirements.txt
```

### 2. Настройка переменных
Скопируйте файл:
```bash
cp .env.example .env
```

### 3. Миграции и запуск
```bash
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

## Docker-вариант
```bash
docker compose up --build
```

## URL'ы
- Главная: `/`

- Личный кабинет: `/dashboard/`
- Панель оператора: `/operator/`
- API: `/api/`
- Админка Django: `/admin/`
